# @hardened/rules-schema

> **Reserved for v3. Not implemented yet.**

This package will host rules that enforce canonical naming across layers (Postgres columns → ORM models → API DTOs → frontend types) and auto-rewrite drifted identifiers.

The initial target stack is Postgres + Prisma + TypeScript + Next.js API routes. Expansion depends on v1 and v2 traction.

See [`docs/vision.md`](../../docs/vision.md) for the roadmap.
